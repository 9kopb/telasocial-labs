This documentation showcases an overview of TagVisor Library project with demonstration, video examples, and pointer to project resources. 

## Introduction

TagVisor is an open-source JavaScript project, and library, that supports the crreation of zoom-based presentations and other visual effects. The script can be attached to a web page and it will transform the page in a visual narrative, thus enabling creative productions based in existing web pages. TagVisor was primarily developed to support web-based digital signage solution — it came of [http://www.telasocial.com](TelaSocial project).
